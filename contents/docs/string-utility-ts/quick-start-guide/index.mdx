## Overview

Welcome to the `Quick Start Guide` for String Utility TS, a lightweight and efficient string manipulation library designed for JavaScript and TypeScript developers. This guide will walk you through the basic setup, installation, and usage of the library, helping you integrate it quickly into your projects.


## Examples

### 1. Reverse String

Reverses the characters in the string.

```jsx showLineNumbers {8}
import { reverseString } = from 'string-utility-ts'

const str = 'Hello World'

const result = reverseString(str)
console.log(result) // dlroW olleH
```

### 3. Starts With 

Checks if the string starts with the specified substring.

```jsx showLineNumbers {8}
import { startsWith } = from 'string-utility-ts'

const str = 'Hello World'

const result = startsWith(str, 'Hello')
console.log(result);    // true
```

### 4. Ends With 

Checks if the string ends with the specified substring.

```jsx showLineNumbers {8}
import { endsWith } = from 'string-utility-ts'

const str = 'Hello World'

const result = endsWith(str, 'Hello')
console.log(result);    // false
```

### 5. Capitalize 

Capitalizes the first letter of the input string and converts the rest to lowercase.

```jsx showLineNumbers {8}

const str = 'Hello World'

const result = capitalize(str)
console.log(result);    // Hello world
```

Removes HTML from the String.

